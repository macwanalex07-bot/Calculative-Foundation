# 📊 Calculative Foundation Project

## 📌 Overview
This project applies Linear Algebra concepts such as vectors, matrices, eigenvalues, PCA, and LDA to analyze student performance data.

## 🎯 Objectives
- Represent data using vectors and matrices
- Perform dot, cross product, norms
- Apply matrix operations
- Compute eigenvalues and eigenvectors
- Implement PCA and LDA

## 📂 Dataset
Student scores across subjects:
- Math
- Physics
- Chemistry

## ⚙️ Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## 📈 Results
- PCA reduced dimensions from 3 → 2
- LDA classified students into categories
- Eigenvalues showed variance distribution
